## Usage

### Install Dependencies

```
$ cd mi-browser

# using yarn
$ yarn
```

### Use it

```
# development mode
$ yarn dev

## production build

# All Platforms
$ yarn build:all

# Windows
$ yarn build:win32
or
$ yarn build:win64

# Mac
$ yarn build:mac

# Linux
$ yarn build:linux
```
